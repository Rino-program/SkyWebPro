# Troubleshooting

## ログインできない

- ハンドルに `@` を付けていないか確認する。
- 通常パスワードではなくアプリパスワードを使用する。
- ログイン画面の「接続診断」を実行し、到達可否を確認する。

## 401/403 が表示される

- セッション期限切れの可能性があります。再ログインしてください。
- Chat API の 401/403 は未認証状態では想定内です。

## 検索結果が不安定

- 連続入力時は前回検索を中断する仕様です。
- 検索文字列を確定後に Enter で再検索してください。

## 画像が自動表示されない

- 設定の「画像自動読み込み」が「Wi-Fi時のみ」の場合、通信種別によって手動読み込みになります。
- 投稿カードの「画像を読み込む」ボタンを押してください。

## 通知が来ない

- 設定の「通知更新間隔」を確認する。
- ネットワーク制限環境では通知APIが遮断される場合があります。

## 表示が崩れる

- ブラウザを最新版に更新する。
- キャッシュクリア後に再読み込みする。
- 拡張機能（スタイル変更系）を一時的に無効化して確認する。
