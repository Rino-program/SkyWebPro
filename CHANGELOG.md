# Changelog

All notable changes to this project are documented in this file.

## [Unreleased]

### Added

- 投稿スレッドの返信を段階的に展開する表示（初期表示 + もっと見る）。
- 通知カードに対象投稿の文脈プレビュー表示。
- 同一投稿・同一通知種別の通知集約表示（アクター展開対応）。
- 通知更新間隔の設定UI（15秒〜5分）。
- トースト表示時間の設定UI。
- 起動時タブ挙動の設定（自動最適化 / 前回タブ）。
- 設定のエクスポート / インポート（JSON）。
- 画像自動読み込み条件設定（常時 / Wi-Fi時のみ）。
- ショートカットキーのユーザー定義（1文字キー、重複防止）。

### Changed

- モーダルの開閉処理を共通化し、Escで最上位モーダルを閉じる動作を統一。
- 検索処理で前回リクエストを中断し、古い結果が上書きしないよう改善。
- 投稿失敗時の下書き退避とユーザー向けエラー表示を改善。

### Fixed

- 返信ツリーの縦線位置ずれを調整。
- 通知アクター名リンクの分離表示を修正。
- 画像アップロード失敗時の再試行（画像単位）を追加し失敗耐性を向上。
