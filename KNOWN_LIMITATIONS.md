# Known Limitations

## 認証とセッション

- セッション情報はブラウザストレージ依存です。
- 共有端末での常用には向きません。

## ブラウザ/ネットワーク依存

- 一部の Safari 環境や組織ネットワークでは API 到達が不安定になることがあります。
- CORS ポリシーにより診断結果が限定される場合があります。

## 機能面

- 画像アップロードは1枚あたり 1MB 制限です。
- 大量データ時の仮想スクロールは未導入です。
- ショートカットのナビゲーション先キー（h/n/s/d/l/p/t）は現状固定です。

## 運用上の注意

- API 仕様変更の影響を受ける可能性があります。
- 大規模変更前には設定のエクスポートを推奨します。
