# Browser Support

SkyWebPro の推奨サポート範囲です。

| Browser | サポート状態 | 備考 |
| --- | --- | --- |
| Chrome (最新) | ✅ 推奨 | フル機能想定 |
| Edge (最新) | ✅ 推奨 | フル機能想定 |
| Firefox (最新) | ✅ 推奨 | 一部API挙動差に注意 |
| Safari (最新) | ✅ 推奨 | 環境によって接続診断が制限される場合あり |
| iOS Safari (最新) | ✅ 可能 | 省電力/通信制約下で挙動差あり |
| Android Chrome (最新) | ✅ 可能 | 端末依存で描画差あり |

## 最低要件

- JavaScript 有効
- HTTPS 配信
- localStorage 利用可能

## 注意点

- 組織ネットワークやフィルタ環境では API 到達性が低下する可能性があります。
- ブラウザ拡張のスタイル干渉で見た目が崩れることがあります。
